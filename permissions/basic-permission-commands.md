# Linux Permission Command

Linux permissions control who can read, write, and execute files or directories

## ls -l

### Description
Displays Detailed file oermissions.

### Example Output
```bash
-rw-r--r-- 1 user 1200 May 12 notes.txt
```

## chmod
### Description
Changes file or directory permission.

### Example
```bash
chmod 755 notes.txt
```

## chmod +x
### Description
Makes a file executable.
### Example
```bash
chmod +x notes.txt
```
